
# Better Call The Animator — Portfolio Website

This is a simple responsive portfolio website for hosting on **GitHub Pages**.

It is designed for:
- 3D Animators
- Motion Artists
- VFX Artists

---

# Folder Structure

better-call-the-animator/

index.html
style.css
script.js
images/profile.jpg

---

# 1. How to Change Your Photo

Replace the image file:

images/profile.jpg

You can rename it but then update the line in index.html:

<img src="images/profile.jpg">

---

# 2. How to Edit Your About Text

Open:

index.html

Find this section:

<!-- EDIT ABOUT TEXT HERE -->

Replace the paragraph with your own description.

---

# 3. How to Add Your YouTube Videos

Find:

VIDEO_ID_HERE

Example:

https://www.youtube.com/embed/abc123xyz

Replace the VIDEO_ID_HERE with your video ID.

Example final code:

<iframe src="https://www.youtube.com/embed/abc123xyz"></iframe>

---

# 4. How to Change Project Titles

Find:

Project Title 1
Project Title 2
Project Title 3

Replace them with your animation names.

---

# 5. How to Change Contact Information

Scroll to the CONTACT section in index.html and edit:

Email
Instagram
GitHub

---

# 6. How to Change Colors

Open:

style.css

Main accent color:

#ff5b2e

Change it to any color you like.

---

# 7. How to Deploy on GitHub Pages

1. Create a GitHub repository
2. Upload all files
3. Go to

Settings → Pages

4. Select:

Deploy from branch

5. Choose:

main / root

Your website will appear at:

https://username.github.io/repository
